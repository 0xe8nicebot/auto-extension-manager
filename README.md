## 主要特点

- 简便直观地管理你的 Chrome 扩展
- 通过配置灵活的规则，来自动打开或关闭扩展

核心功能定位是可以通过配置规则来自动打开或关闭扩展，而且规则可以配置得足够灵活。后期也会根据反馈，持续完善规则的设置。
